# Trend Engine

Statistical analysis library for golf shot trend detection and insight generation.

## Installation

```bash
pip install trend-engine
```

## Development

```bash
python -m venv .venv
.venv\Scripts\activate  # Windows
pip install -e ".[dev]"
```

## Testing

```bash
pytest
```

## Linting

```bash
ruff check src tests
ruff format src tests
pyright
```
